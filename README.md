Crowdsourced Emergency Alert System

A real-time public safety platform that enables users to report, verify, and receive emergency alerts instantly
 based on location and incident severity.

Live Demo🔗:-https://prayag1010.github.io/Crowdsafe/
_________________________________________________________________________________________________________________

Overview:-

The Crowdsourced Emergency Alert System is designed to improve public safety through community-driven emergency reporting and real-time notifications.

Users can instantly report incidents such as:
- Road accidents
- Fires
- Floods
- Medical emergencies
- Natural disasters
- Public safety threats

The system shares alerts with nearby users and responders, helping reduce emergency response time and improve situational awareness.
_________________________________________________________________________________________________________________

Key Features:-

Emergency Reporting
- Report incidents in real time
- Add emergency type and details
- Automatic timestamp generation

Location Tracking
- Detect user location using Geolocation API
- Display incidents on interactive maps

Instant Alerts
- Notify nearby users immediately
- Real-time emergency broadcasting

Crowdsourced Verification
- Community-based incident confirmation
- Reduce false alerts and misinformation

Interactive Dashboard
- Live incident visualization
- Emergency status tracking

Responsive Design
- Mobile-friendly UI
- Clean and modern interface

How It Works:-

1. User reports an emergency incident
2. System captures user location
3. Incident data is stored in database
4. Nearby users receive instant alerts
5. Responders monitor incidents in real time
6. Community members verify or update reports
